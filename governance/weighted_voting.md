# Trust-Weighted Voting

This document explains the governance model for Pi Hybrid Token using trust-weighted voting.

## Voting Weight
Voting weight is calculated based on:
1. **Token holdings** — number of tokens owned
2. **Staking duration** — how long tokens have been staked
3. **Ecosystem contributions** — activity, participation, verified transactions
4. **Reputation score** — based on previous actions, fraud-free record

## Rules
- Each participant’s vote is multiplied by their total weight
- Quorum must be met for decisions to be valid
- Malicious behavior reduces voting weight automatically
