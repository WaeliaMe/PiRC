# Profit Sources & Verification

This document describes the sources of profit in the Pi Hybrid Token model and how they can be verified on-chain.

## Sources of Profit
- Revenue from token project usage
- Transaction fees within the ecosystem
- Payments made by projects utilizing Pi tokens
- Other decentralized revenue streams

## Verification
- All profit transactions are recorded on-chain
- Smart contracts automatically calculate allocations for:
  - Liquidity pools
  - Developers
  - Ecosystem reserve
- Transparency ensures all participants can validate profit distribution
