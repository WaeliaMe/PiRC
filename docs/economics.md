# 📊 Economics Model – PIRC Hybrid Launch

## Overview
The PIRC Hybrid Launch model introduces a **mathematically grounded token economy** designed to ensure:

- Fair value distribution  
- Market stability  
- Resistance to manipulation  
- Long-term sustainability  

Unlike traditional supply-based models, this system is **value-driven**, anchored by real activity and enforced through deterministic constraints.

---

## 🔹 Core Value Model (IPPR)

### Internal Purchasing Power Reference (IPPR)

The system separates:

- **External Value** → Speculative (CEX / IOU pricing)  
- **Internal Value** → Deterministic and utility-based  

We define:
IPPR = Function(Liquidity, Verified Transactions, System Activity)

The IPPR is anchored primarily by:

- 💧 **65% Liquidity Pool Allocation**
- 📈 Verified on-chain transactions

---

## 🔹 Transaction-Based Value Scaling

Project value increases based on verified activity:
ProjectValue = PiBaseValue × (1 + RealTxCount / TotalTxThreshold)

### Example:
- PiBaseValue = 314,159  
- RealTxCount = 5,000  
- TotalTxThreshold = 10,000
ProjectValue = 314,159 × (1 + 0.5) = 471,238.5

📌 This ensures:
- Real usage drives value  
- Fake volume has limited impact (if filtered properly)

## 🔹 Φ (Phi) Solvency Constraint

### Definition

The system enforces a strict solvency invariant:
Φ = Liquidity / Outstanding Credit Supply

### Rule

If Φ < 1.0:
Reward distribution is halted
Reserve redistribution is paused
System expansion is blocked

### Purpose

- Prevents over-expansion  
- Ensures collateralization  
- Protects ecosystem stability
