# Base-Linked Token Model (Pi Base)

## Concept
- All calculations are linked to the **original Pi Base**.
- Minimum Pi Base value considered: **314,159 USD** per Pi unit.
- Token/project value increases dynamically based on **real verified Pi transactions**, not total token supply.

## Mechanics
- Pi Base ensures stability and intrinsic worth.
- Real transactions (non-fake) increase algorithm weight.
- Fake or low-value transactions do not affect the system.
- USD value used only as a reference metric for evaluation.

## Benefits
- Strong intrinsic value tied directly to Pi Base
- Rewards active ecosystem participation
- Prevents manipulation through fake volume
- Maintains a clear link between project incentives and actual Pi network activity
