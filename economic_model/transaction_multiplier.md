# Transaction Multiplier & Value Scaling (Pi Base)

- Each verified Pi transaction contributes to **value growth based on Pi Base**.
- Multiplier formula example:
ProjectValue = PiBaseValue * (1 + RealTxCount / TotalTxThreshold)

- Encourages real, high-value Pi transactions
- Prevents spam/fake transaction inflation
- Rewards participants proportionally to verified Pi activity
- All calculations reference Pi Base, USD only for display/evaluation purposes
