# Token Allocation Pseudo-Code

This example shows how profits from the token project can be allocated:

```python
# Profit allocation example
total_profit = 1000000  # Profit from token project
liquidity = total_profit * 0.65
developers = total_profit * 0.20
reserve = total_profit * 0.15

print("Liquidity:", liquidity)
print("Developer allocation:", developers)
print("Reserve:", reserve)
