# Profit Flow Architecture

## Overview
This document explains how profits generated by token projects flow through the hybrid token model.

## Profit Flow

1. Project generates revenue
2. Revenue is recorded or verified
3. Profit distribution rules are executed
4. Funds are allocated according to the model

## Allocation

Profit is distributed as follows:

- 65% → Liquidity Pools
- 20% → Developers
- 15% → Ecosystem Reserve

## Reserve Redistribution

Over time, the reserve gradually decreases and is redistributed:

15% → 10% → 5%

This strengthens liquidity and rewards long-term development.
